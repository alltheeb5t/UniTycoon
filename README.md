# UniTycoon
 
Part of the ENG1 Assessment 2 deliverables for alltheeb5t (Cohort 3 Group 5).

Contains the latest build of UniTycoon, licenced under Apache 2.0.

## Running

### Generating Jar file

```
./gradlew lwjgl3:dist 
```
```
gradlew.bat lwjgl3:dist
```
will generate a jar file with the required assets inside:

Location:
``lwjgl3/lib/UniTycoon-x.x.x.jar``

### Running the game

```
./gradlew lwjgl3:run
```
```
gradlew.bat lwjgl3:run
```

This will launch the game from build files
